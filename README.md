# Expense Tracker CLI

A command line application to track your daily expenses built with Node.js.

## Features
- Add a new expense
- View all expenses
- Delete an expense by ID
- View total amount spent

## Tech Stack
- Node.js
- fs module

## How to Run
node index.js add "Title" Amount "Category"
node index.js view
node index.js delete <id>
node index.js total
